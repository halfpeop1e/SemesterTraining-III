# 轨道线路与信号控制 —— 模块交接文档

> **模块**：轨道线路与信号控制（线路建模 + MA 计算 + 联锁仿真 REST + 周期胶水）
> **负责人**：黄豪
> **后端包**：`com.bjtu.railtransit.signal`
> **前端页**：`frontend-control-center/src/pages/LineSignal/`
> **文档版本**：2026-07-11（对接闭环 G1–G5/G8/G4 + A4 后同步）
> **读者**：总控(周)、车载(郭)、仿真集成、其他 agent —— **如何对接本模块**
> **更细算法/需求**：仓库外 `.docs/信号模块-需求说明书.md` v1.6；引擎契约 `docs/signal/MA模块与仿真引擎对接接口说明.md`

**一句话**：给定全线状态 + 全车状态，为每列车算出 **EoA（`endOfAuthorityM`）+ 限速（`maxSpeedKmh`）**；不确定则 **fail-safe 收紧**。角色 ≈ RBC/TCC，不是完整 CBI，也不是车载 ATP。

**指令优先级**：`安全 ATP > 信号 MA > 总控调度 > ATO 建议`

## 0. 给对接方 30 秒

| 你是谁 | 你该做什么 |
|--------|------------|
| **总控 / 仿真** | 每 tick 调 `SignalCycleService.runCycle(trains, nowSec)`（仓库已接）；进路用 **trainId 绑定**（`/route/assign` 或 `build?trainId=`）；MA **权威读 Registry**（`/ma/latest`），勿用简算覆盖 |
| **车载 / HMI** | 读 `GET /api/signal/train/{id}/snapshot` 的 `movementAuthority*`；上报 `POST /api/signal/report/status`（可带 P2：故障限速/定位/完整性）；降级时 DMI 哨兵见 §3.5 |
| **联锁操作 / 前端** | 道岔/进路/信号/TSR 一律走 `/api/signal/*`；与 `GET /line` **同一 LineProfile 缓存** |
| **只算 MA 的调用方** | 同进程 `MovingAuthorityService.compute(...)`；或 `POST /api/signal/ma` |

**生产数据流（已接通）**：

```
SimulationService tick
  → SignalCycleService.runCycle(runtimeTrains, nowSec)
      → 同步 activeRoutes（G1）
      → refreshAxleOccupancy（G2）
      → TrainStateAdapter（P2 + loadFactor）
      → MovingAuthorityService.compute(line, trains, activeRoutes, nowSec)
      → MovementAuthorityRegistry.replace
      → applyAtpConstraint 回写 runtime 限速/EoA/EB
      → MA 降级写 SignalEventLog（G4）
  → atpSafetyEnforce：Registry 有车则不覆盖信号 MA（G8）
```

## 3. 对外接口

### 3.1 同进程：MA 计算

```java
Map<String, MovingAuthority> compute(LineProfile line, List<TrainState> trains,
                                     Map<String, Route> routes, double nowSec);
// routes / nowSec 可省略的重载见 MovingAuthorityService
```

- **纯函数**（不改 line/trains/routes 引用内容的算法语义；联锁改的是共享 LineProfile 上的 aspect/state）
- 返回 `Map<trainId, MovingAuthority>`
- 字段：`endOfAuthorityM`(m) `maxSpeedKmh` `basis` `event` `capSignalId` `routeId` `timestamp`
- **`event` 为 `DEGRADED` / `MA_EXPIRED` / `POSITION_LOSS` → 立即停车**（`maxSpeedKmh`≈0）

**生产路径请优先**：`SignalCycleService.runCycle`（已含 Adapter、计轴刷新、Registry、ATP 回写），不要每模块自己拼一套。

### 3.2 同进程：周期与注册表

| API | 含义 |
|-----|------|
| `runCycle(Iterable<domain.TrainState> runtimeTrains, double nowSec)` | 每 tick 入口 |
| `MovementAuthorityRegistry.snapshot()` / `get(trainId)` | **权威 MA** |
| `assignRoute(trainId, Route)` / `getActiveRoutes()` | 车→进路（与 REST 绑定一致） |

### 3.3 REST（前缀 `/api/signal`，统一 `ApiResponse`）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/line` | 全量 `LineProfile`（**与联锁同一缓存实例**） |
| POST | `/ma` | body: lineProfile + trains + routes? + nowSec? |
| GET | `/ma/latest` | Registry 快照（仿真权威 MA） |
| GET | `/status` | onlineTrains / alertCount / health / simulationTime / generation / source |
| GET | `/events?limit=` | **真实事件流**（环形 200；新在前） |
| POST | `/switch/{id}/operate?state=` | `NORMAL` / `REVERSE` / `FAIL` |
| GET | `/switch/all` | |
| POST | `/route/build?routeId=&trainId=` | trainId **可选**；有则 build 成功后再绑车 |
| POST | `/route/{routeId}/cancel` | 取消站场进路 **并** 清相关绑定 |
| GET | `/route/built` | |
| POST | `/route/assign?trainId=&routeId=` | 仅绑定（route 须已 built） |
| POST | `/route/unassign?trainId=` | 仅解绑 |
| GET | `/route/bindings` | `Map<trainId, routeId>` |
| POST | `/signal/{id}/set?aspect=` | 灯色 |
| GET | `/signal/aspects` | |
| POST | `/tsr` | body: startM, endM, speedLimitKmh, active? |
| DELETE | `/tsr/{tsrId}` | |
| GET | `/tsr` | |
| POST | `/report/status` | 车载状态 → StatusFusion（可带 P2 字段） |
| GET | `/train/{trainId}/snapshot` | HMI：train + MA + commands + timetable + 通信/安全 |

### 3.4 编解码工具

- `Direction.fromDriverConsole(int)`：`0` 上 / `1` 下 / 其余 INVALID
- `SignalAspect.decodeSignalSystem(int)` / `isProceed()`（**仅 GREEN**）/ `toDriverConsoleScreen()`
- `SwitchState.fromProtocol` / `toProtocol`：`0x01` 定位 / `0x02` 反位 / `0x04` 四开 FAIL
- `LineProfile.locateMileage(segId, offsetCm)` 等坐标查询

### 3.5 TrainState 构造（权威入口）

```java
// 段相对 + m/s
TrainStateAdapter.fromSegmentRuntime(line, trainId, segId, offsetM, speedMps, dir, lengthM, ts);

// 绝对里程
TrainStateAdapter.fromAbsolute(trainId, positionM, speedKmh, dir, lengthM, ts);

// + P2
TrainStateAdapter.fromAbsolute(..., faultSpeedLimitKmh, positionLost, integrityLost);

// + P2 + 载重 loadFactor（A4；NaN=未知→按额定制动）
TrainStateAdapter.fromAbsolute(..., fault, posLost, intLost, loadFactor);
```

- **禁止**各模块自建第二套 TrainState 转换
- 非法 segId → `positionM=NaN` → `DEGRADED`

### 3.6 无效哨兵合同（打包 DMI 时，车载侧）

当 `event ∈ {DEGRADED, MA_EXPIRED, POSITION_LOSS}`：

| 字段 | 哨兵 |
|------|------|
| 速度 16bit | `0xFFFF` |
| 距离 24/32bit | `0xFFFFFFFF` |

MA 对象本身仍是数值域：`maxSpeedKmh=0`，`endOfAuthorityM=当前位置`。

1. `computeOne` fail-safe 顺序与六维候选 + UP/DOWN 的 min/max

## 6. 本模块需要别人喂什么

| 输入 | 提供方 | 形式 | 接线状态 |
|------|--------|------|----------|
| 列车位/速/向/长 | 仿真 runtime | 每 tick | ✅ `runCycle` |
| `nowSec` | 引擎时钟 | 与 timestamp 同基准 s | ✅ |
| **进路（按车）** | 总控或本页操作 | `activeRoutes`：`trainId → Route` | ✅ REST assign/build?trainId= |
| Signal.aspect | 联锁 REST / 默认 | enum | ✅ |
| Switch.state | 联锁 REST | NORMAL/REVERSE/FAIL | ✅ |
| 计轴 occupied | 本模块 Cycle 按车位刷 | — | ✅ 自动（G2） |
| TSR | REST `/tsr` | LineProfile.tsrs | ✅ |
| faultSpeedLimitKmh | 车载 StatusReport | km/h，NaN=无 | ✅ 链路通，靠上游填 |
| positionLost / integrityLost | 车载 | bool | ✅ 同上 |
| loadFactor | runtime 客流 | 0~1，NaN=未知 | ✅ A4 |
| LineProfile 静态 | Loader | 已 mileage index | ✅ 自给 |

### 6.1 signal 域 `TrainState` 字段

| 字段 | 必填 | 单位 | 备注 |
|------|------|------|------|
| trainId | ✅ | | |
| positionM | ✅ | m 车头 | |
| speedKmh | ✅ | km/h | |
| lengthM | ✅ | m >0 | |
| direction | ✅ | UP/DOWN/INVALID | |
| timestamp | 带 nowSec 时 | s | 过期阈值 **1.6s** |
| accelerationMps2 | ⚪ | m/s² | 不参与约束 |
| faultSpeedLimitKmh | ⚪ | km/h | NaN=不参与 min |
| positionLost / integrityLost | ⚪ | bool | true→POSITION_LOSS |
| loadFactor | ⚪ | 0~1 | NaN=额定制动 |

### 6.2 进路双层模型（对接必懂）

| 层 | 存储 | 含义 | MA 是否读 |
|----|------|------|-----------|
| 站场进路 | `builtRoutes` | 灯/冲突/可取消 | 否 |
| 车进路 | `activeRoutes` | 哪列车走哪条 | **是**（key=`trainId`） |

规则摘要：

- `build` 不带 trainId → 只办站场
- `build` 带 trainId → build 成功后再 assign；**assign 失败不回滚 build**
- 一车一条（覆盖）；一路一车（冲突拒绝）；禁止自动猜车
- `cancel` 服务端清该 route 的所有绑定
